SWS to ICS Converter

How to use

Open the timetable app from the Student Portal (eg the page here) and build a timetable. Ensure you select the "Full Trimester" option for weeks.

Once you are at the view timetable page, right click the page and select your "View Source" option (Chrome shows this as "View Page Source", other browsers may vary). Select everything (Ctrl+A), and paste it as is into the above text box, and hit Go.

Known Issues

  • Exporting modules with multiple events happening on the same day at the same time (or even an overlap) will break the exporter
I take no responsibility if the data for an event is incorrect when exporting from this application. If you use this application you should ensure the data is accurate for what you exported.
